# Velicor UI

A modern, high-performance log ingestion and analysis dashboard.

## Features
- **Real-time Telemetry:** Stream logs directly from your microservices.
- **Archive Search:** Powerful filtering and keyword search for historical logs with spinner-less status code entries.
- **Instant Service Switcher:** Search and switch between active service logs directly from the header using a custom dropdown.
- **Granular Webhook Setup:** Target and isolate webhook notifications to specific microservices using a searchable multi-select control.
- **Inline Fleet Controls:** Rotate ingestion credentials and change log retention duration settings directly inside the service dashboard cards.
- **Polished Dark Mode:** High-contrast color-coded log lines, expandable JSON payloads, and premium carbon/slate aesthetics.

## Tech Stack
- **Framework:** React 19 (TypeScript)
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide React
- **Routing:** React Router 7

## Getting Started
1. `npm install`
2. `npm run dev`
