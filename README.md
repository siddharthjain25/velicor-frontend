# Velicor UI

A modern, high-performance log ingestion and analysis dashboard.

## Features
- **Real-time Telemetry:** Stream logs directly from your microservices.
- **Archive Search:** Powerful filtering and keyword search for historical logs.
- **Traffic Simulator:** Stress test your ingestion pipeline.
- **Fleet Management:** Provision and manage multiple microservice keys.

## Tech Stack
- **Framework:** React 19 (TypeScript)
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide React
- **Routing:** React Router 7

## Getting Started
1. `npm install`
2. `npm run dev`
